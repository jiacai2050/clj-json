## What is JSON?

```
value |
    string
    number
    true/false/null
    object
    array

string |
    ""
    "chars"    


char |
    char
    char chars    

number |
    number 
    number number

object |
    {}
    {pairs}        

pairs |
    string : value    
    string : value, pairs

array |
    []
    [members]    

members |
    value    
    value, members
```
